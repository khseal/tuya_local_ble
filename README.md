# Unlock Token (`unlock_token`)

Для BLE‑разблокировки hc7n0urm в `devices.json` нужен **8‑значный** параметр
`unlock_token` (тот же, что формирует приложение Tuya/Smart Life для этой сессии ключа).

## Как получить токен из HCI‑лога (скрипт в репозитории)

1. **Включите снятие HCI** на Android: *Настройки разработчика* → *Bluetooth HCI snoop log*.
2. Откройте приложение замка, выполните открытие по BLE, затем выключите лог и **скопируйте** файл
   (часто `btsnoop_hci.log` из `/sdcard/Android/data/...` или через `adb pull`).
3. Установите **Wireshark** (нужна утилита командной строки **`tshark`**; на Windows она в каталоге Wireshark).
4. Установите зависимости Python: `pip install pycryptodome`
5. Запустите из корня репозитория (или скопируйте скрипт к себе):

   ```bash
   python3 extract_unlock_token_from_btsnoop.py /путь/к/btsnoop_hci.log "ВАШ_LOCAL_KEY"
   ```

   В качестве `local_key` укажите **полный** ключ из Tuya Cloud (как в `devices.json`).  
   Для BLE используются **первые 6 символов** для вычисления `login_key` (как в интеграции).

6. В выводе ищите строку **`unlock_token:`** с восемью цифрами. Пропишите её в `devices.json`.

**Важно:** в логе ATT‑кадры могут быть разрезаны по HCI; скрипт опирается на **tshark**, который собирает полные ATT‑значения. Сырой разбор файла без Wireshark часто **не** находит токен.

**JSON:** между полями обязательны **запятые**. Пример ошибки — пропущенная запятая после `"product_name"` перед `"unlock_token"`.

Общие сведения про HCI snoop:  
https://stackoverflow.com/questions/28445552/bluetooth-hci-snoop-log-not-generated

# Установка

Поместите папку интеграции в `custom_components`:

    config/
    └── custom_components/
        └── tuya_local_ble/
        
А файл конфигурации:

    config/
    └── tuya_local_ble/
        └── devices.json

После этого перезапустите **Home Assistant**.
В **Настройки** → **Устройства и службы** → **Добавить интеграцию** выберите **Tuya Local BLE**
(или дождитесь обнаружения по Bluetooth). Нужен корректный `devices.json` и запятые в JSON.
